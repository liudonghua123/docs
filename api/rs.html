教职工列表查询

请求参数 ​

名称位置类型必选说明
depquerystring否单位
dqztdmquerystring否当前状态代码
yrfsdmquerystring否用人方式代码

返回示例

成功

json
{
  "data": [
    {
      "XM": "XM",
      "ZGH": "ZGH"
    },
    {
      "XM": "XM",
      "ZGH": "ZGH"
    }
  ],

» ZGHstringtruenone职工号none
